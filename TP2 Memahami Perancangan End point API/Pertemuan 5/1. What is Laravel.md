# Penjelasan Laravel dan RESTful API

## 1. Apa itu Laravel?

Laravel adalah framework PHP modern yang dirancang untuk membuat aplikasi web dengan mudah dan cepat. Framework ini menyediakan berbagai tools dan fitur yang memudahkan developer dalam mengembangkan aplikasi web yang scalable dan maintainable.

### Karakteristik Laravel:
- **Bahasa**: Dibangun dengan PHP
- **Pola Arsitektur**: MVC (Model-View-Controller)
- **Database ORM**: Eloquent ORM untuk interaksi database yang mudah
- **Routing**: Sistem routing yang powerful dan intuitif
- **Migrations**: Kontrol versi untuk database schema
- **Blade Templating**: Template engine yang powerful

### Keuntungan Laravel:
1. **Mudah dipelajari** - Dokumentasi lengkap dan komunitas besar
2. **Produktivitas tinggi** - Mempercepat development dengan fitur built-in
3. **Keamanan** - Proteksi terhadap SQL injection, XSS, dan CSRF
4. **Scalable** - Cocok untuk aplikasi kecil hingga enterprise
5. **Testing** - Built-in testing tools (PHPUnit)

### D. Langkah-Langkah Download dan Instalasi Laravel

#### Menggunakan Composer (Recommended)

```bash
# 1. Buka Command Prompt / Terminal
# 2. Navigasi ke folder yang ingin menyimpan project
cd C:\laragon\www\  # Windows (jika menggunakan laragoin)

# 3. Buat project Laravel baru
composer create-project laravel/laravel nama-project

# 4. Masuk ke folder project
cd nama-project

# 5. Jalankan server
php artisan serve
```


### E. Verifikasi Instalasi Laravel

```

├── tests/                # File testing

├── .gitignore

├── composer.json         # Composer configuration

└── package.json          # NPM configuration (jika ada)
```

### G. Troubleshooting Umum

| Masalah | Solusi |
|---------|--------|
| `Composer not found` | Pastikan Composer sudah diinstall dan di-add ke PATH |
| `PHP not found` | Pastikan PHP sudah diinstall dan di-add ke PATH |
| `SQLSTATE Error` | Periksa file `.env` dan pastikan database sudah dikonfigurasi |
| `Port 8000 already in use` | Gunakan port lain: `php artisan serve --port=8001` |
| `Permission denied` | Jalankan command prompt sebagai Administrator |

## 4. MVC Pattern di Laravel

### Model
- Merepresentasikan struktur data
- Berinteraksi dengan database
- Contoh: `User.php`, `Product.php`

### View
- Menampilkan data kepada user
- File template (Blade)
- Contoh: `users.blade.php`, `product.blade.php`

### Controller
- Menghandle business logic
- Menerima request dan mengirim response
- Menghubungkan Model dan View
- Contoh: `UserController.php`, `ProductController.php`

## 5. Routing di Laravel

Routing adalah proses mengarahkan HTTP request ke controller yang sesuai.

### Contoh Routes (routes/web.php):

```php
// Route GET
Route::get('/users', 'UserController@index');

// Route POST
Route::post('/users', 'UserController@store');

// Route PUT/PATCH
Route::put('/users/{id}', 'UserController@update');

// Route DELETE
Route::delete('/users/{id}', 'UserController@destroy');

// Route dengan parameter
Route::get('/users/{id}', 'UserController@show');
```
