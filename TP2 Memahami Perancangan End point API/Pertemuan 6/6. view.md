# View (Sederhana)

View di Laravel adalah tampilan (UI) yang dikembalikan ke client. Biasanya menggunakan Blade template engine yang berada di `resources/views`.

- File Blade berakhiran `.blade.php`, misal `resources/views/posts/index.blade.php`.
- Mengirim data dari Controller ke View:

```php
return view('posts.index', ['posts' => $posts]);
// atau
return view('posts.index', compact('posts'));
```

- Contoh sederhana di Blade:

```blade
<!-- resources/views/posts/index.blade.php -->
@extends('layouts.app')

@section('content')
	<h1>Posts</h1>
	@foreach($posts as $post)
		<article>
			<h2>{{ $post->title }}</h2>
			<p>{{ Str::limit($post->content, 150) }}</p>
		</article>
	@endforeach
@endsection
```

- Keamanan: gunakan `{{ }}` untuk escape output (mencegah XSS). Jika memang perlu menampilkan HTML, pakai `{!! $html !!}` dengan hati-hati.
- Layout & partials: gunakan `@extends`, `@section`, `@include` untuk struktur yang rapi.
- Kontrol alur: gunakan direktif Blade seperti `@if`, `@foreach`, `@can`, `@auth`.
- Render JSON (API): controller bisa mengembalikan `return response()->json($data);` atau menggunakan API Resource.
