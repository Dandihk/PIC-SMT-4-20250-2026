## 4. Buat function register untuk membuat akun di authcontroller
regiister digunakan untuk membuat akun dan membuat token auth
```php
public function register(Request $request)
    {
        //validator untuk cek adata yang masuk sesuai requirment atau tidak
        $validator = Validator::make($request->all(), [
            'name' => 'required|string|max:255', 
            'email' => 'required|string|email|unique:users', //harus unik tidak boleh sama 
            'password' => 'required|string|min:8'
        ]);

        if ($validator->fails()) return response()->json($validator->errors(), 422); //error jika validator salah

        $user = User::create([  //memasukan imputan ke tabel user
            'name' => $request->name,
            'email' => $request->email,
            'password' => Hash::make($request->password), //meng enskripsi password dengan hash
        ]);

        $token = $user->createToken('auth_token')->plainTextToken; //membuiat token auth

        return response()->json(['data' => $user, 'access_token' => $token, 'token_type' => 'Bearer'], 201); //menampilkan pesan sukses
    }
```
use model user, hash dan validator
```php
use App\Models\User;
use Illuminate\Support\Facades\Hash;
use Illuminate\Support\Facades\Validator;
```
## 4. Buat function login untuk membuat akun di authcontroller
login digunakan untuk memvalidasi user yang akan digunakan login, ketika salah akan menampilkan error
```php
public function login(Request $request)
    {
        $user = User::where('email', $request->email)->first(); //cek inputan email ada yang sama tidak seperti di tabel user amnil data pertama 

        //jika tidak ada email dan password yang sama maka menampilkan error
        if (!$user || !Hash::check($request->password, $user->password)) { 
            return response()->json(['message' => 'Email atau password salah'], 401);
        }

        //membuat token auth untuk login
        $token = $user->createToken('auth_token')->plainTextToken; // men
        //menampilkan pesan sukses
        return response()->json(['message' => 'Login sukses', 'access_token' => $token, 'token_type' => 'Bearer']);
    }
```
## 5. Buat function login untuk membuat akun di authcontroller
logout digunakan untuk menghapus token yang sedang di gunakan
```php
public function logout(Request $request)
    {
        // Menghapus token yang sedang digunakan
        $request->user()->currentAccessToken()->delete();
        return response()->json(['message' => 'Berhasil logout']);
    }
```
## 6. Buat exception untuk handling sesuatu yang tidak terduga 
exception adalah kondisi kesalahan yang terjadi saat program mencoba melakukan operasi yang tidak valid
(misal membuka post data tapi belum login)
handling ini bisa dibuat didalam ***bootstrap/app.php*** cari function ***withExceptions***
```php
// Untuk custom pesan unauthenticated
        $exceptions->render(function (AuthenticationException $e, Request $request) {
            if ($request->is('api/*')) {
                return response()->json([
                    'success' => false,
                    'message' => 'Maaf, Anda belum login. Silakan login terlebih dahulu.',
                ], 401);
            }
        });
```


## 7. Buat route todo dan kategori
untukroute api amaka penempatannya beda dari route biasanya ya itu di taruh dalam routes/api.php
route api resource dibnungkus dengan route middleware untuk membegal ditengah jalan untuk cek autrntifikasinya 
jadi fungsi midelware seperti palang pintu perumahan, ketika kamu ingin masuk perumahan harus di cek identitasnya dahulu 
kalau punya dan benar boleh masuk kalo tidak tidak boleh masuk.
```php
// Route Protected (Harus Login)
Route::middleware('auth:sanctum')->group(function () {
    Route::post('/logout', [AuthController::class, 'logout']);

    Route::apiResource('kategori', KategoriController::class);
    Route::apiResource('todo', TodoController::class);
    });

// Route Public (Bisa diakses tanpa login)
    Route::post('/register', [AuthController::class, 'register']);
    Route::post('/login', [AuthController::class, 'login']);
```
